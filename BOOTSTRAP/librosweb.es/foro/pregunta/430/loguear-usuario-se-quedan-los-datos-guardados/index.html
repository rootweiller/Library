¿Exactamente que quieres hacer? Porque para cerrar sesión, sólo debes acceder a la ruta que colocas en la configuración logout de tu firewall

Ahora si lo que quieres es evitar que se vea el formulario de login si el usuario ha iniciado sesión, puedes hacer algo como:

public function loginAction()
{
    if ($this->isGranted("IS_AUTHENTICATED_FULLY")) {
        // si está logueado, redirigimos
        return $this->redirectToRoute("homepage");
    }
 
    // ...
}

Para que no se vea el formulario de login al acceder a su URL, sino que te envía a la página por defecto que ven los usuarios logueados.

#5

Buenas,

Si lo que deseas es forzar el cierre de la sesión al ir a la página de login, podrias hacer algo como:

public function loginAction()
{
    if ($this->isGranted("IS_AUTHENTICATED_FULLY")) {
        // si está logueado, cerramos sesión
        return $this->redirectToRoute("logout"); //ruta que usas para cerrar sesión
    }
 
    // ...
}

Además deberás hacer que el logout mande a la página de login nuevamente, esto ultimo lo logras en la configuración del componente de seguridad:

# Tu Firewall
    logout:
       path: logout_path
       target: login_path
